# Python 自动化运维脚本集合

## 项目简介
为学习 Python 自动化运维、数据采集而开发的脚本集合，主要用于服务器监控、网页数据采集等场景，匹配自动化运维岗位需求。

## 脚本说明
1. **server_monitor.py**
   - 功能：获取服务器系统版本、主机名、CPU核心数、内存使用率、磁盘使用率等硬件信息
   - 用途：辅助日常运维巡检，监控服务器资源使用情况，保障系统稳定运行

2. **web_crawler.py**
   - 功能：基础网页数据采集，模拟提取新闻标题与链接等结构化数据
   - 用途：可扩展为业务数据汇总、竞品信息监控、内部系统数据对接等工具

## 技术栈
- Python 3.x
- 依赖库：psutil（服务器信息获取）、platform（系统信息获取）

## 学习目标
通过实践掌握 Python 在自动化运维、数据交互场景中的应用，提升解决实际问题的能力，希望能快速适配岗位工作需求。
